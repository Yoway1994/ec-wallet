## 用戶故事
作為 電商平台的顧客，
我想要 在完成訂單後獲得一個專屬的 BSC 收款地址，
以便 我可以使用加密貨幣完成訂單支付。

## 描述
建立一個 API 端點，允許電商系統在創建訂單後請求一個 BSC 鏈收款地址。系統將從地址池中分配一個可用的 BSC 地址，並將其與訂單編號關聯，然後返回該地址以供顧客支付使用。

## 開發任務
- [ ] 設計並實現 `/api/v1/payment-address` RESTful API 端點

## 驗收標準
- [ ] 每個訂單編號必須對應唯一的一個 BSC 地址
